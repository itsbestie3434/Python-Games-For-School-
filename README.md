# Python-Games-For-School-
a list of 100 (or more) python games using pure python and some packages.
### How to play the python games
use some sort of compiler or ide, like the python built in IDE, or a platform like PyCharm.
### If your game has an error
If you come across an error, check the text file inside the game files as it will say what packages need to be installed.
Or check if you have python installed to the latest update.
If the problem still persists, make an issue pointing it out and i will fix it.
